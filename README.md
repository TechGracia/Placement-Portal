# Placement Portal

## About the Project
Placement Portal is a web-based platform designed to streamline the placement process for students and recruiters. Built using **PHP, MySQL, HTML, and CSS**, this project was developed as a **final-year BCA project**.

## Features
- **Student Module**:
  - Register and manage profiles
  - Apply for job postings
  - Track application status
- **Recruiter Module**:
  - Post job openings
  - Manage applications and candidates
- **Admin Panel**:
  - Oversee student and recruiter activities
  - Manage job postings and user roles

## Technologies Used
- **Front-end**: HTML, CSS
- **Back-end**: PHP
- **Database**: MySQL

## Installation Guide
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/Placement-Portal.git
   ```
2. Set up a local server using XAMPP or WAMP.
3. Import the `database.sql` file into MySQL.
4. Configure database connection in `config.php`.
5. Start the server and access the portal via `localhost`.
